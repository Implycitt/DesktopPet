## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Implycitt/DesktopPet.git
   ```
2. Run the run.bat script 
   ```sh
   ./run.bat
   ```
Conversely clicking on the run script will start the program as well
